var声明的变量可以作用在全局作用域里面
var  a=1  var a=2后面可以覆盖前面的




var在全局作用域范围内都有效
let只在局部作用域范围内有效，块级里面才有效


    var   a=[];
    for  (var  i=0;i<10;i++){
        a[i]=function () {
            console.log(i);
        };

    }
        a[6]();//括号是执行后面的方法,打印出最后的结果出来10
/*
讲解一下，原因，var声明的变量在全局都有效，所以当循环每一次的时候，i的值都会发生改变
所以结果是10，let是块级，在小范围是有效的


 */

结果：6


如果for循环是var定义的变量的话，那么结果就是10，每一次循环这个值都会发生改变，i是全局作用域，所以结果是10
