main.js入口文件
如果是在package.json里面引入的包，不需要加入路径
--save是保存到当前的项目里面去

npm  run  dev  启动程序的执行
http://localhost:8000/project/select这个是浏览器的路劲 

main.js是启动文件，当你启动的时候，就可以执行相对应的app.vue文件
